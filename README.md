# simple-chat-app

Simple Chat App

## Getting Started

A simple chat app that works like messenger

### Tech Used

MongoDB, Express, Angular, Node.js, Socket.io

### Running the App

1. `cd` to `server` and run the command

```
npm install
...
npm start
```

1. `cd` to `client` and run the command

```
npm install
...
ng serve --open
```

Now, the app should be running. Enjoy!
